# Circular Buffer - Assignment 1

## 📌 Description

This project implements a Circular Buffer (Ring Buffer) using the C programming language.

The program demonstrates how data can be written to and read from a fixed-size buffer using circular indexing.



## ⚙️ Features

* Initialize the buffer
* Insert (write) characters into the buffer
* Remove (read) characters from the buffer
* Detect and handle:

  * Buffer Overflow
  * Buffer Underflow



## ▶️ Program Flow

1. The user enters their name.
2. The string `"CE-ESY"` is appended to the name.
3. The full string is written into the circular buffer.
4. The data is then read back and printed to the screen.



## 🧪 Testing

The program runs two tests:

### 🔴 Test 1: Small Buffer

* Buffer size is smaller than the input string.
* This causes **Overflow**.
* Some characters cannot be inserted.

### 🟢 Test 2: Large Buffer

* Buffer size is large enough.
* All characters are stored and printed correctly.
